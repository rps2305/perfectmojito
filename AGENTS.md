# Perfect Mojito - Website Project

## Project Context
- Dutch-language website about Cuban Mojitos
- Modern, colorful design aesthetic
- Header image: `bikini-girl-beach-bar-tropical-drinks.webp`

## Tech Stack
- Tailwind CSS (latest version)
- Standard modern HTML/CSS/JS setup

## Key Commands
- Dev server: `npx @tailwindcss/cli -i ./src/input.css -o ./dist/output.css --watch`
- Build: `npx @tailwindcss/cli -i ./src/input.css -o ./dist/output.css --minify`
- Preview: Open `dist/index.html` in browser (no server needed, uses relative paths)

## Design Direction
- Tropical/beach bar theme
- Vibrant, summery color palette (mint greens, rum browns, lime yellows, white)
- Dutch language throughout
- Fun, inviting atmosphere befitting a cocktail website
